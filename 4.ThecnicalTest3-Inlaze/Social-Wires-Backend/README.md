## Installa Packages

```bash
$ npm install
```

## Lift Database With Docker
```
$ docker-compose up -d
```

## Swagger Endpoints
```
http://localhost:3000/api/
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
